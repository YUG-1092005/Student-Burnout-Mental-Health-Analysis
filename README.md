# 📊 Student Burnout & Mental Health Analysis (SQL + Power BI)

## 📌 Project Overview
This project presents an end-to-end data analysis of student burnout and mental health using SQL for data processing and Power BI for visualization.

The goal is to identify relationships between stress, anxiety, depression, lifestyle habits, and academic factors that contribute to burnout and dropout risk.

---

## 🎯 Objectives
- Analyze student burnout trends
- Identify key factors affecting mental health
- Study impact of academic pressure and lifestyle
- Build an interactive dashboard for insights

---

## 🛠️ Tech Stack
- SQL (MySQL)
- Power BI
- CSV Dataset
- Data Analysis Techniques (Aggregation, Correlation, Segmentation)

---

## 🗄️ SQL Analysis

### Database Setup
- Created `Student_Burnout` database
- Designed `students` table with academic, mental health, and lifestyle attributes

### Data Processing
- Imported data using `LOAD DATA INFILE`
- Performed aggregation and grouping
- Created KPIs using SQL queries

### Key SQL Insights
- Burnout distribution across risk levels
- Stress, anxiety, depression by categories
- Dropout risk analysis
- Lifestyle impact (sleep, activity, screen time)

---

## 📊 Power BI Dashboard

### KPI Metrics
- Avg Burnout Score: **1.78**
- Avg Health Index: **7.03**
- Avg Stress Level: **4.24**
- Avg Sleep Hours: **6.50**
- Dropout Risk: **1.41%**

## 📈 Key Insights

- High stress leads to higher burnout and dropout risk  
- Financial stress significantly increases burnout  
- Physical activity improves mental health  
- Screen time (~5 hrs) is above ideal levels  
- Exam pressure is high and contributes to burnout  
- Strong correlation between stress and anxiety  
- Burnout is linked with dropout risk
